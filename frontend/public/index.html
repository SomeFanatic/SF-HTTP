How to use Igor's HTTP Client

  1. Download the request template.
  2. Edit the JSON file to match your API request.
  3. Import the JSON using Import button.
  4. Select request from the list.
  5. Click Send to execute.

Requests are processed in-memory only. No data is stored on the server.
